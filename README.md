# Welcome to my project
An Online Examination Platform System is a web-based application designed to conduct exams digitally. It allows administrators to create exams, manage questions, and evaluate results, while students can securely take tests from anywhere.
<br>
<br>
# Features<br>
🔐 User Authentication (Admin & Student login/signup) <br>
📝 Create & Manage Exams<br>
📚 Question Bank (MCQs, descriptive questions) <br>
⏱️ Timer-based Exams <br>
📊 Automatic Result Generation<br> 
📈 Performance Analysis <br>
🧾 Exam History Tracking <br>
🌐 Responsive UI <br>
